Введение в HTML

HTML (от англ. HyperText Markup Language – язык гипертекстовой разметки) представляет собой язык разметки, который используется для создания веб-страниц. Он позволяет добавлять на страницу текст, изображения, ссылки, таблицы и другие элементы.

Основные элементы HTML

Каждый HTML-документ состоит из элементов, которые заключены в теги. Теги определяют тип содержимого, например, заголовок, параграф, изображение, ссылка и т.д.

Некоторые основные теги HTML:

  • <html> - определяет начало HTML-документа.
  • <head> - содержит метаданные документа.
  • <title> - определяет заголовок документа.
  • <body> - содержит основное содержимое документа.
  • <h1> - заголовок первого уровня.
  • <p> - параграф.
  • <a> - гиперссылка.
  • <img> - изображение.
  • <ul> - неупорядоченный список.
  • <ol> - упорядоченный список.
  • <li> - элемент списка.
  • <table> - создает таблицу
  • <tr> - создает строку таблицы
  • <th> - создает ячейку заголовка таблицы
  • <td> - создает ячейку таблицы
