Введение

JavaScript (JS) — это мультипарадигмальный язык программирования, используемый для создания веб-приложений и веб-сайтов. Он позволяет создавать динамические эффекты, взаимодействовать с пользователем и обрабатывать данные, введенные на странице.

Основы языка

JS поддерживает множество конструкций и методов для работы с данными и элементами страницы. Вот некоторые из них:

  • переменные
  • условные операторы
  • циклы
  • массивы
  • функции
  • объекты

Примеры кода

Ниже приведены примеры кода на JavaScript:

// Отобразить время на момент загрузки страницы в элементе с идентификатором #datetime
let now = new Date();
let datetimeElement = document.getElementById("datetime");
datetimeElement.innerHTML = now.toLocaleString();

Элемент #datetime

// Скрипт, который скрывает/показывает элемент при клике на кнопку:
const toggleBtn = document.getElementById('toggle-btn')
const text = document.getElementById('text')
toggleBtn.addEventListener('click', () => {
    text.hidden = !text.hidden
});

Пример текста для скрытия/отображения.


// Скрипт для кнопки, которая увеличивает счетчик на 1 при каждом нажатии:
const counter = document.getElementById('counter');
const button = document.getElementById('button');
let count = 0;

button.addEventListener('click', () => {
    count++;
    counter.textContent = `Счетчик: ${count}`;
});
Счетчик: 0
